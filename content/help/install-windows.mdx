## Download

1. Go to [voicetypr.com/download](/download)
2. Click **Download for Windows**
3. The `.exe` installer downloads to your Downloads folder

## Install

1. Run the `.exe` installer
2. Follow the setup wizard
3. VoiceTypr installs to `C:\Program Files\VoiceTypr`

## First Launch

Windows Defender may show a warning about an unrecognized app.

1. Click **More info**
2. Click **Run anyway**

## Grant Permissions

### Microphone Access

1. Open **Settings > Privacy & Security > Microphone**
2. Toggle **Microphone access** and **Let apps access your microphone** to On
3. Find VoiceTypr in the list and toggle it On

### Keyboard Hook (Optional)

For global hotkeys to work across all apps, VoiceTypr may need to run with elevated permissions. If your hotkey doesn't work in some apps:

1. Right-click the VoiceTypr shortcut
2. Select **Run as administrator**
