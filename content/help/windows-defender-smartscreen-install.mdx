## Why this happens

SmartScreen sometimes warns about newer or less widely downloaded apps.

That warning does not automatically mean the installer is malicious.
It means Windows wants you to confirm that you intended to run it.

## Fix it

1. Double-click the VoiceTypr installer
2. When the blue SmartScreen dialog appears, click **More info**
3. Click **Run anyway**
4. Continue through the installer

## If you only see a browser warning

If Chrome, Edge, or another browser blocks the file before you run it:

1. Keep the download
2. Open your **Downloads** folder
3. Run the `.exe` manually from there

## If it still refuses to launch

Check:

- whether your security software quarantined the installer
- whether the file finished downloading completely
- whether you are launching the latest installer from [voicetypr.com/download](/download)

If needed, delete the installer and download it again.

## What to do after install

Once VoiceTypr is installed:

- enable microphone access in Windows settings
- test dictation in a simple app like Notepad
- then move to Cursor, Gmail, or another target app
