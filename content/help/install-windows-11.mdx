Use this guide if you are on **Windows 11**.

## 1) Download and Run Installer

1. Open [voicetypr.com/download](/download)
2. Click **Download for Windows**
3. Run the downloaded `.exe`
4. If UAC appears, click **Yes**

## 2) Handle SmartScreen Warning

If Microsoft Defender SmartScreen appears:

1. Click **More info**
2. Click **Run anyway**

## 3) Enable Microphone Permissions (Windows 11)

Go to **Settings > Privacy & security > Microphone** and enable:

1. **Microphone access**
2. **Let apps access your microphone**
3. **Let desktop apps access your microphone**

Restart VoiceTypr after changing permissions.

## 4) First Dictation Check

1. Open any text field (Notepad, browser, editor)
2. Trigger VoiceTypr hotkey
3. Speak a short sentence
4. Confirm text appears where your cursor is

If text does not appear, see [Troubleshooting](/help/troubleshooting).
