Use this guide if you are on **Windows 10**.

## 1) Download and Install

1. Open [voicetypr.com/download](/download)
2. Click **Download for Windows**
3. Run the downloaded `.exe`
4. Complete the setup wizard

## 2) Defender / SmartScreen Prompt

If Windows warns about an unrecognized app:

1. Click **More info**
2. Click **Run anyway**

## 3) Enable Microphone Permissions (Windows 10)

Go to **Settings > Privacy > Microphone** and turn on:

1. **Allow access to the microphone on this device**
2. **Allow apps to access your microphone**
3. **Allow desktop apps to access your microphone**

Restart VoiceTypr.

## 4) Hotkey Permission Edge Case

If VoiceTypr hotkey fails in an app opened as Administrator, run VoiceTypr as Administrator too.

## 5) Verify First Launch

1. Open a text field
2. Use your VoiceTypr hotkey
3. Speak and confirm text appears

If audio is not detected, re-check microphone settings and test your input device in Windows Sound settings.
