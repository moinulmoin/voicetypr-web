# VoiceTypr for Windows dictation

VoiceTypr is a Windows voice typing and dictation app for users who want private speech-to-text across the desktop. It is built for normal text fields: Cursor, VS Code, Word, Gmail, Slack, Notion, Linear, ChatGPT, Claude, support tools, browser forms, and other places where a cursor accepts pasted text.

## Windows positioning

Windows includes built-in voice features, but many users still need a dictation-first workflow for long prompts, docs, emails, and notes. VoiceTypr is positioned as a modern local dictation tool rather than a full computer-control system.

## Best for

- Windows users replacing deprecated Windows Speech Recognition habits.
- People who use Windows Voice Typing for short snippets but need a daily long-form workflow.
- Buyers comparing Dragon Professional, Microsoft Dictate, Windows Voice Access, and local AI dictation.
- Developers, founders, support teams, and AI power users who dictate prompts and replies.

## Not for

- Full hands-free Windows control. Windows Voice Access is the better fit for command/navigation control.
- Meeting transcription, speaker labels, or shared transcript archives.
- Users who only need occasional dictation inside Microsoft Office and are satisfied with built-in tools.

## Trust points

- Supports Windows 10+ and macOS 13+.
- VoiceTypr transcribes your voice on your machine by default after setup.
- Optional AI formatting sends text only if the user enables it.
- The app uses a simple hotkey: hold to speak, release to insert text.
- It is not browser-only and does not require writing inside a proprietary editor.
- Lifetime pricing starts at $39 once, with a 3-day free trial.
